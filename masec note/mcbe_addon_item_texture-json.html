アドオン作成 - item_textures.jsonについて

目次は右上の「≡」から！

　このファイルはブロックのテクスチャに関するファイルで、リソースパックのtexturesフォルダに一つだけあります。

━ ┳ manifest.json
　 ┣ pack_icon.png
　 ┗ textures
　 　 ┗ item_textures.json
機能

　アイテムにどんなテクスチャを貼るかを決定するための中間行程です。
　主な役割として、テクスチャとTerrainIDを結びつけます。

アイテム本体
　↓　minecraft:iconコンポーネントで紐づけ
TerrainID
　↓　item_texture.jsonで紐づけ
テクスチャ
？
TerrainIDについては後に説明します
ファイル内容

　本来は、テクスチャを紐付けしたいアイテムすべてについて書きますが、はじめは例として一つのアイテムの場合を示します。

Res
textures/item_textures.json
{
  "resource_pack_name": "vanilla",
  "texture_name": "atlas.terrain",
  "texture_data": {
    "apple": { "textures": "textures/items/apple" }
  }
}

　概形はこんな感じです。
　自分で新しく作る場合はこれ↑をコピペするといいでしょう。

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": {
"apple": { "textures": "textures/items/apple" }
} }

　この赤線で囲った部分が一つのカタマリです。これを適当に「ユニット」と呼ぶことにします。
(このサイト独自の呼び方です)

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": { "apple": { "textures": "textures/items/apple" } } }

　上のこの部分に関しては、変えても特におもしろいことは起きないので説明を省きます。コピペしたらそのままでいいと思います。

TerrainID

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": {
"apple": { "textures": "textures/items/apple" }
} }

　便宜上、このマーカー部分をTerrainIDと呼びます。(このサイト独自の呼び方です(けっこう言います))
　そしてこれは各自で自由に決められます。
　本来ふさわしくない呼び方ですが、ブロックのterrain_texture.jsonと似た書き方をするため、そこでの呼び方を流用しています

テクスチャを指定

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": {
"apple": { "textures": "textures/items/apple" }
} }

　ここにテクスチャのファイルの場所(ファイルパス)を指定します。

指定するアイテムを増やす

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": {
"apple": { "textures": "textures/items/apple" },
"apple_golden": { "textures": "textures/items/apple_golden" }
} }

　複数のブロックについて記述する場合は、ユニット自体を増やします。
　ユニットとユニットをつなぐときのカンマ( , )は忘れるとエラーになります。

発展的な内容

複数のテクスチャ指定

{ "resource_pack_name": "vanilla", "texture_name": "atlas.terrain", "texture_data": {
"bucket": { "textures": [ "textures/items/bucket_empty", "textures/items/bucket_milk", "textures/items/bucket_water", "textures/items/bucket_lava", "textures/items/bucket_cod", "textures/items/bucket_salmon", "textures/items/bucket_tropical", "textures/items/bucket_pufferfish", "textures/items/bucket_powder_snow", "textures/items/bucket_axolotl" ] }
} }

　もともとデータ値をもっているアイテムならば、上から順にデータ値0,1,2,...のテクスチャになります。
　また、別のところで何番目か選択できる場合はこのように書いて選択肢を持たせます。
